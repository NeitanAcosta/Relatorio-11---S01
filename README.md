# Relatório 11 - S01

## Descrição

Este repositório contém a resolução dos exercícios propostos na atividade da disciplina, abordando conceitos de Programação Orientada a Objetos em Java.

Os exercícios exploram os seguintes tópicos:

* Classes abstratas
* Herança
* Polimorfismo
* Interfaces
* Composição
* Agregação
* ArrayList
* HashMap
* Sobrescrita de métodos

## Estrutura

Cada exercício foi desenvolvido em um projeto separado do IntelliJ IDEA, contendo suas respectivas classes e arquivos de apoio.

## Ambiente de Desenvolvimento

* Java OpenJDK 25
* IntelliJ IDEA
* Git/GitHub

## Uso de Inteligência Artificial

Foi utilizada a ferramenta ChatGPT como apoio para consulta, revisão de código e esclarecimento de dúvidas durante o desenvolvimento da atividade.

Conforme solicitado no enunciado, cada exercício contém um arquivo `IA_REPORT.md` descrevendo a utilização da ferramenta, os prompts utilizados e os aprendizados obtidos durante o processo.

### Observação

Um dos arquivos `IA_REPORT.md` apresenta pequenas diferenças em relação aos demais devido a atualizações realizadas durante o desenvolvimento da atividade e à evolução das interações com a ferramenta. Apesar disso, todos os relatórios descrevem corretamente o processo utilizado na resolução dos exercícios.
